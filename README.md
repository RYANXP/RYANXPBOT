## Clone this project

```bash
> git clone https://github.com/RYANXP/RYANXPBOT
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> cd RYANXPBOT
> bash install.sh
```

### Usage
```bash
> npm start
```
